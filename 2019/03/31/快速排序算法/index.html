快速排序算法

快速排序算法稳定性：不稳定。
快速排序算法复杂度：平均时间O(nlogn),最坏情况O(n²),适用于乱序的序列。
C++代码手写实现从小到大快速排序

#include <stdio.h>

#include
using namespace std;
//输入数据函数 参数：数组a，元素的个数n
void input(int a[],int n)
{
int i=0;
for(;i<n;i++)
{
a[i]=rand();//随机生成数放入数组a中
}
}
//显示数据函数 参数: 数组a, 元素个数n
void output(int a[],int n)
{
int i=0;
for(;i<n;i++)
{
cout<<a[i]<<”\t”; //输出元素
}
cout<<endl;//换行
}
//快速排序算法函数 参数:数组a,开始的下标start,结束的下标end
void quickSort(int a[],int start,int end)
{
int i,j=0;//定义i,j下标
int key=0;//定义基准值
i=start;j=end;
key=a[i];
//判断start是否大于end，若不大于进行排序，大于则退出函数
if(start>=end)
{
return;
}
//判断循环动态下标i是否大于j,若大于则将将key基准值赋值于下标为i的数组
while(i<j)
{
//从后往前找比key小的值，若找到则将找到的值存放在下标为i的数组中且i下标+1,若找不到则将j下标-1
while(i<j&&a[j]>key)//注意：要有i<j的约束条件不然会造成i>j的后果将有重复值出现
{
j–;
}
if(i<j)
{
a[i]=a[j]；
i++;
}
//从前往后找比key大的值，若找到则将找到的值存放在下标为j的数组中且j下标-1，若找不到则将i下标+1
while(i<j&&a[i]<key)//注意：要有i<j的约束条件不然会造成i>j的后果将有重复值出现
{
i++;
}
if(i<j)
{
a[j]=a[i];
j–;
}
}
//当i=j时将key基准值赋值于下标为i的数组
a[i]=key;
//按照前面的方法递归比key小的部分
quickSort(a,start,i-1);
//按照前面的方法递归比key大的部分
quickSort(a,i+1,end);
}
//主函数实现相关方法
int main()
{
int n=0;//定义元素个数变量
cout<<”请输入所需元素的个数”<<endl;
cin>>n;//给元素个数变量赋值
//定义数组a
int a[20]={0};
//为数组赋值（原序列）
input(a,n);
//显示原序列
cout<<”原序列”<<endl;
output(a,n);
//实现快速排序算法
quickSort(a,0,n-1);//开始下标为0，结束下标为元素个数-1
//显示快速排序后的序列
cout<<”快速排序后的序列”<<endl;
output(a,n);
return 0;
}
/*
总结：
1.思想:分治法,选取基准值来进行筛选，分成小于基准值的部分和大于基准值的部分，再将其两部分用同样的方法进行递归。冒泡算法的扩展
2.注意:时刻清楚i,j值下标的位置，不能让i<j的值。若等于则将基准值赋予i下标的数组值。
3.时间复杂度：平均时间O(nlogn),最坏情况O(n²),使用于多和杂的序列。
